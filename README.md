# Page Replacement Simulator

Desktop simulator for OS paging algorithms with a clean UI and step-by-step results.

## Features

- FIFO, LRU, and Optimal algorithms
- Tabbed output per algorithm
- Step-by-step frame transitions with **HIT/FAULT**
- Summary tiles: total requests, best algorithm, best fault count

## Run (Windows / PowerShell)

```bash
javac PageReplacementSimulator.java
java PageReplacementSimulator
```
