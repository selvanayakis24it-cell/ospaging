What you can do

  • Choose algorithms: FIFO, LRU, Optimal (run any combination).
  • Compare quickly: shows total requests and best algorithm.
  • Trace every step: frames update per reference with HIT/FAULT status.

Algorithms (quick notes)

FIFO LRU Optimal
FIFO
Replace the oldest page in memory.
LRU
Replace the least recently used page.
Optimal
Replace the page used farthest in the future.

How to run

  1. Ensure Java is installed: java -version
  2. Compile: javac PageReplacementSimulator.java
  3. Run: java PageReplacementSimulator
